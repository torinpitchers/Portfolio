Work

MyUniPal:


MyUniPal is a companion app for students that provides 'tools' that are useful for students. The application was developed in xcode and written in Apples new Swift. Utilising the newly developed HotDeals REST API developed by ukhotdeals.com to provide users with the latest and best deals from the whole of the uk. The finance manager was designed to be smart and informative but to be non intrusive, allowing you full control to go over budget and if so then recalculating your daily, weekly, and mothly allowance.

The application was developed for the purpose of assessment for a university module and as such the requirements of the project were different to a usual client. The project development time was also quite short, 6 weeks to be exact. As a result the application was not released to the app store and is current stored away in my project stash.

Sentiment Analysis Modelling


This project was for my final project during my degree at Coventry University and concerns developing a natural language processing application based on a sentiment analysis model, using the NL tool kit (NLTK). The purpose of this project was to investigate the modeling of a sentiment analysis system that can accurately predict the polarity of text written reviews, learning from existing consumer reviews paired with a star rating. Then use this system to predict the polarity of new unseen reviews and determine the accuracy and error rate. This was performed by investigating aspects of the model and then determine how the particular aspect affected the accuracy of the overall model, using this information to build the 'perfect' model with highest possible accuracy.

The application was written in python and uses the machine learning classifier Naive bayes to train and then classify the data. This project was successful and I achieved a 90% accuracy when predicting a yelp reviews positivity. The report is available here
